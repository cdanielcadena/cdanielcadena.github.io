# EvolVert preservation build

This folder is a static mirror of the live Weebly site. The visible page text has not been rewritten or summarized.

## What was changed

- Repaired malformed background-image URLs generated by Weebly.
- Added the homepage background image (`2118629476.png`) locally.
- Added `.nojekyll` so GitHub Pages serves the mirrored files exactly as stored.
- Added `CNAME.example` for later use with `evolvert.com`.

## GitHub test deployment

1. Create a public repository named `cdanielcadena.github.io`.
2. Upload everything inside this folder to the repository root.
3. In **Settings → Pages**, deploy from `main`, folder `/ (root)`.
4. Test at `https://cdanielcadena.github.io`.
5. Do not configure `evolvert.com` until every page, image, PDF, and external embed has been checked.
